📱 Инструкция и сохранение файла

📲 Вариант А: Использование сайта (PWA на экран Домой)
  1. Откройте Safari на вашем iPhone.
  2. Перейдите по ссылке: https://misha12333211-ctrl.github.io/2Chat/
  3. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх).
  4. Выберите «На экран "Домой"» ➔ «Добавить».
📁 Вариант Б: Сохранение локального файла в «Файлы» (для работы офлайн)
  1. Скачайте `.html` файл чата на iPhone (через Telegram/Safari).
  2. Нажмите на файл и выберите «Поделиться» ➔ «Сохранить в "Файлы"».
  3. Выберите папку «На iPhone» и нажмите «Сохранить».
  4. Откройте стандартное приложение 📁 Файлы на iPhone и нажмите на этот HTML-файл — он запустится в браузере с полной функциональностью локального PWA.
⚙️ Правило парной настройки никнеймов

Никнеймы связывают каналы шифрования. У вас и собеседника они должны быть записаны **зеркально** (только латиницей):

📍 Вариант установки:
• Вы: Ваше имя = Stas | Имя собеседника = Masha
• Собеседник: Ваше имя = Masha | Имя собеседника = Stas

Пароль шифрования комнаты у обоих должен быть строго 1234.

💬 Статусы доставки и работа без сети
  • ⏳ Песочные часы: Сообщение сохранено локально на устройстве (ждет появления сети).
  • ✓ (1 галочка): Зашифровано и доставлено в Яндекс Диск.
  • ✓✓ (2 зеленые галочки): Прочитано собеседником.
⚠️ Важно: Локальный файл из приложения «Файлы» полностью сохраняет базу сообщений даже при перезапуске, но синхронизация с Яндекс Диском будет происходить только при активном подключении к интернету.
